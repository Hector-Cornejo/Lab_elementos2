Comparación de microcontroladores

Héctor Cornejo Morales y Bruno Carballido Rosas
Ing. Mecatrónica
Fecha: Febrero 20, 2025
Arduino Electrónica

- Resumen -

La practica de esta semana consistió en realizar 3 diferentes códigos para probarlos en 3 diferentes microcontroladores. Primero se realizaría una simulación de los circuitos y una ves terminada la prueba se construirían los circuitos en físico. El objetivo era observar si notábamos alguna diferencia en el desempeño a la hora de cargar el código o llevar a cabo el funcionamiento de los circuitos.

- Introducción -

Un microcontrolador es un circuito integrado programable, básicamente es un chip que funciona como una minicomputadora que está diseñada para administrar tareas específicas en sistemas integrados sin tener un sistema operativo complejo. Cada microcontrolador cuenta con una arquitectura diferente y varían en cuestión de desempeño, componentes y utilidades.

Un microcontrolador cuenta con elementos esenciales que definen su arquitectura.

  1. Unidad Central de Procesamiento (CPU)
    • Es el cerebro de la computadora, es el encargado de procesar datos y ejecutar instrucciones.
    • La CPU puede ser de 8, 16, 32 o 64 bits, con esto se refiere a la cantidad de datos que el microcontrolador o computadora puede manejar al mismo tiempo.
  2. Memoria
    • Cada computadora cuenta con diferentes tipos de memoria
    • Flash ROM: Almacena datos de forma permanente (hasta que se cambien por otro programa).
    • RAM: Almacena datos de forma temporal en lo que se corre el programa.
    • EEPROM: Almacena datos en el microcontrolador incluso cuando este esta apagado.
  3. Periféricos o Pines
    • Los pines son modulos de conexión que permiten la interaccion con otros electronicos o un circuito. Estan los GPIOs (pines de entrada y salida),PWM, timers, I2C, etc.
  4. Frecuencia
    • La frecuencia es basicamente la velocidad con la que el procesador ejecuta instrucciones, una mayor frecuencia es igual a mejor rendimiento pero de la misma manera consume mas energía. Hay microcontroladores que tienen una frecuencia establecida unica, esto se refiere a que no puedes modificar el valor de su freceuncia interna, pero puedes aumentarla usando algun componente externo. Por otro lado hay otros microcontroladores a los cuales les puedes cambiar el valor de su frecuencia interna sin la necesidad de algun componente externo.

- Materiales -

Para esta practica vamos a usar 3 microcontroladores:

  • ESP32-C6
  • Arduino UNO
  • Attiny85

Una protoboard, un pulsador, un LED, cables, una resitencia de 10KOhms y otra de 5KOhms
